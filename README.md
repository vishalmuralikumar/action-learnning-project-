## Installation (Local)

To run this project open terminal and run this code.

```bash
  pip install -r requirements.txt
```

After Successful Installing, Do this step to run FlaskAPI

```bash
  python main.py
```

To run the WebApp, type the below code

```bash
  streamlit run frontend.py
```
